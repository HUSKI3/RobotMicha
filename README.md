# Features
RobotMicha supports the following:

- 70% Coverage of the guidelines
- Code suggestions for fixes
- Nice TUI
- Suggestions
- Easily expandable rule-set

# Installation
```bash
$ pip3 install loguru rich
```

# Usage
```bash
$ python3 robotmicha web/index.html
```
